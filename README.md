# pyopt

[Live Demo](https://pyoptimize.streamlit.app)

Stock portfolio optimization for the Vietnamese market using Modern Portfolio Theory.

Built with Streamlit, PyPortfolioOpt, and Riskfolio-lib. Fetches live data from the Vietnam stock exchange via vnstock.

## Features

- **Efficient Frontier Analysis** вЂ” Max Sharpe, Min Volatility, and Max Utility portfolios with interactive scatter plot of 5,000 random portfolios
- **Hierarchical Risk Parity (HRP)** вЂ” Clustering-based allocation with dendrogram visualization
- **Discrete Allocation** вЂ” Convert portfolio weights into actual share counts for a given VND investment amount
- **Excel Report Export** вЂ” Downloadable `.xlsx` reports generated by Riskfolio-lib
- **Risk Analysis** вЂ” Risk metrics table, drawdown analysis, and return distribution plots

## Quickstart

Requires Python 3.10+.

```bash
# Install dependencies
uv sync

# Run the app
uv run pyopt
```

Or without the CLI entry point:

```bash
uv run streamlit run streamlit_app.py
```

## Configuration

### Sidebar Controls

| Control | Description |
|---|---|
| Ticker symbols | Multi-select from all HOSE/HNX/UPCOM listings |
| Date range | Historical data period for optimization |
| Risk aversion | Parameter for Max Utility portfolio (0.1вЂ“10.0) |
| Colormap | Color scheme for the efficient frontier scatter plot |

### Portfolio Strategies

| Strategy | Objective |
|---|---|
| Max Sharpe | Maximize risk-adjusted return |
| Min Volatility | Minimize portfolio standard deviation |
| Max Utility | Maximize quadratic utility given risk aversion |

### MCP Server (Claude Desktop)

Add to your `claude_desktop_config.json`:

```json
{
  "mcpServers": {
    "pyopt": {
      "command": "uv",
      "args": ["run", "python", "server.py"],
      "cwd": "/path/to/modern-st"
    }
  }
}
```

This exposes 5 tools to Claude: `symbols`, `optimize`, `hrp`, `allocate`, and `risk`.
